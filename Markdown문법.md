# 마크다운 활용법



## 기본 표기 사용방법

* 제목은 #을 통해 레벨을 지정한다.

  샵#의 갯수에 따라 h1~4까지 헤드라인 크기가 적용

* *를 사용해 점자 사용

## 코드 블록

```java
int salary = 5000;
```



* 영문자로 ```를 작성한뒤 java 또는 python과 같은 프로그래밍언어를 붙여서 작성 가능

## 코드강조
