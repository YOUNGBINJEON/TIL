# 23_자바스크립트 100제 - OX 문제

## 문제23 : OX

`console.log(10/3)`의 출력 결과는 3이다.







## 해답

* 3.333333이 나오므로 X

만약 정수로 떨어지게 하고 싶다면 `Math.floor()` 함수를 사용하면 된다.
