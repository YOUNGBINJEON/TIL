# 11_자바스크립트 100제 - for를 이용한 기본 활용

## 문제11 : for를 이용한 기본 활용

1부터 100까지 모두 더하는 Code를 <pass> 부분에 완성하세요. `for`문을 사용해야 합니다.

```js
let s = 0;

//pass

console.log(s);
```



**출력 화면**

```
5050
```





## 해답

```js
let s = 0;

//pass
for(let i=1; i<=100; i++) {
  s += i;
}

console.log(s);
```







### 결과

**코드**

```js
let s = 0;

//pass
for(let i=1; i<=100; i++) {
  s += i;
}

console.log(s);
```
