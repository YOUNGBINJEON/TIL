# 05_자바스크립트 100제 - for문 계산

## 문제5 : for문 계산

다음 코드의 출력 값으로 알맞은 것은?

```js
var a = 10;
var b = 2;

for(var i=1; i<5; i+=2) {
  a += i;
}

console.log(a+b);
```

1) 10
2) 12
3) 14
4) 16



## 해답

i = 1 일 때, a = 10 + 1

i = 3 일 때, a = 11 + 3

a + b = 16









### 결과

**코드**

```js
var a = 10;
var b = 2;

for(var i=1; i<5; i+=2) {
  a += i;
}

console.log(a+b);
```
