# 문제 1 : 배역의 삭제

다음 배열에서 400, 500을 삭제하는 code를 입력하세요.

```js
var nums = [100, 200, 300, 400, 500];
```





## 해답

```js
nums.pop();
nums.pop();
```

`pop` 메서드는 배열에서 마지막 요소를 제거하여 그 값을 호출자(caller)에게 반환한다.

빈 배열의 경우 `undefined`를 반환한다.



### 결과

**출력 코드**

```js
var nums = [100, 200, 300, 400, 500];

nums.pop();
nums.pop();

document.write(nums);
```
