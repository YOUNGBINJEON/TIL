# 22_자바스크립트 100제 - 배수인지 확인하기

## 문제22 : 배수인지 확인하기

다음 중 변수 i가 6의 배수인지 확인하는 방법으로 올바른 것은?

1) i / 6 == 0
2) i % 6 == 0
3) i & 6 == 0
4) i | 6 == 0
5) i // 6 == 0







## 해답

* `/` : 몫을 반환하기에 배수확인 불가
* `%` : 나머지 값으로 반환하기에 정답
* `&` : AND 비트 연산자로 두 개의 피연산자의 각 자리마다 이진법으로 대응하는 비트가 모두 `1`일 경우 `1`을 반환한다.
* `|` : OR 비트 연산자로 두 개의 피연산자의 각 자리마다 이진법으로 대응하는 비트가 둘 중 하나가 `1`일 경우 `1`을 반환한다.
* `//` : 주석 처리를 하는 문법이다.
